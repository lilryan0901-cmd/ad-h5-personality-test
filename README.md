# AD钙奶童年风味人格鉴定 H5互动测试

这是大广赛营销策划中的 H5 互动环节原型。

项目通过轻量人格测试，把“酸甜口感”转化为年轻人的社交人格标签。用户答题后生成童趣人格瓶海报，可保存分享。技术实现为纯 HTML + CSS + JavaScript + html2canvas。

## 功能列表

- 随机题目：从题库中随机抽取 5 道互动题。
- 人格比例计算：计算酸涩值、回甘值、活力值、纯净值。
- 动态人格瓶海报：使用 HTML/CSS 直接生成卡通人格瓶，不依赖本地图片。
- 海报保存：通过 html2canvas 保存结果海报为 PNG。
- 移动端适配：移动端优先，桌面端也可正常浏览。

## 部署方式

- GitHub Pages 主链接：运行 `bash scripts/deploy_all.sh` 后生成。
- Vercel 备用链接：安装并登录 Vercel CLI 后可由部署脚本尝试生成。
- Netlify 备用链接：安装并登录 Netlify CLI 后可由部署脚本尝试生成。

## 本地预览

可以直接双击 `index.html` 打开，也可以运行：

```bash
python3 -m http.server 9000
```

然后访问 `http://127.0.0.1:9000/index.html`。

## 海报保存说明

答题和结果展示不依赖服务器。保存海报功能需要网络加载 html2canvas；无法加载时不影响答题和结果展示，页面会提示“海报保存功能加载失败，请检查网络后重试”。

## 重要声明

- 本项目为竞赛展示原型。
- 不采集用户真实信息。
- 不依赖服务器。
- 不包含真实券码核销系统。
- 不再使用 Flask、Cloudflare Tunnel、ngrok、localhost 或本地产品图片作为最终提交方案。
